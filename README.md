# Pendekatan-Nilai-Fungsi-dengan-Newton-Divided-Difference
Mini project metode numerik menggunakan Interpolasi Newton Divided Difference untuk melakukan pendekatan nilai fungsi pada titik tertentu, yaitu f(0,5). Perhitungan dilakukan menggunakan tabel divided difference dan polinom interpolasi Newton di Microsoft Excel hingga diperoleh estimasi nilai fungsi secara numerik.
